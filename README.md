# Bank AI assistant

## Установка
1. Установить Python 3.12
2. Установить нужные библиотеки
```
pip install -r requirements.txt
```

## Запуск
```
python main.py
```

## Создать .env
```
OPENAI_API_KEY=sk-12345
```

## Стек

 - Python 3.12
 - SQLite Database
 - OpenAI API
 - При работе над проектом использовалась модель GPT 4o и OpenAI Assistants API
